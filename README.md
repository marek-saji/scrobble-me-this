# `@saji/scrobble-me-this`

A command-line tool to scrobble tracks to Last.fm from a CSV input.

## Installation

```sh
npm install -g @saji/scrobble-me-this
```

## Usage

```sh
scrobble-me-this --help
```

E.g.

```sh
scrobble-me-this \
  --api-key YOUR_API_KEY \
  --api-secret YOUR_API_SECRET \
  --username USERNAME \
  --password PASSWORD \
  tracks.csv
```
